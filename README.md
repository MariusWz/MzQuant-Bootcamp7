# MzQuant-Bootcamp7

## 📚 项目简介

**7-Day Intensive Quant-Trading Sprint Plan** - 一个为期7天的量化交易系统搭建实战教程。

本项目通过7天的密集学习，手把手教你从零开始搭建一个完整的量化交易系统，涵盖数据获取、策略开发、回测框架、风险管理到实盘部署的全流程。

## 🎯 学习目标

- 掌握量化交易系统的基础架构设计
- 学习数据获取、清洗和存储的最佳实践
- 实践技术指标计算和策略信号生成
- 构建完整的回测框架和绩效评估系统
- 理解风险管理和仓位管理的核心概念
- 完成一个可运行的量化交易系统

## 🚀 快速开始

### 环境要求
- Python 3.13+
- Jupyter Notebook
- 核心依赖：pandas, numpy, matplotlib, plotly, ta, pyarrow

### 安装步骤
1. 克隆项目
```bash
git clone <repository-url>
cd MzQuant-Bootcamp7
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 启动开发环境
```bash
jupyter notebook
```
